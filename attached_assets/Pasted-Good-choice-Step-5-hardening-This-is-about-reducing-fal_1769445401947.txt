1️⃣ SMOOTH MOVEMENT (reduce jitter)
🔴 Add this helper function (anywhere above your route)
def average_position(frames, key, window=3):
    positions = []
    for i in range(len(frames)):
        if i < window:
            continue
        xs = [frames[j][key][0] for j in range(i-window, i)]
        ys = [frames[j][key][1] for j in range(i-window, i)]
        positions.append({
            "frame": frames[i]["frame"],
            "pos": (sum(xs)/len(xs), sum(ys)/len(ys))
        })
    return positions


This smooths landmark movement across multiple frames instead of trusting one.

2️⃣ REQUIRE SUSTAINED MOVEMENT (no single-frame triggers)
🔴 Replace your movement_start function with this
def movement_start(frames, key, threshold=0.01, sustain_frames=2):
    smoothed = average_position(frames, key)
    count = 0

    for i in range(1, len(smoothed)):
        x1, y1 = smoothed[i-1]["pos"]
        x2, y2 = smoothed[i]["pos"]
        dist = np.sqrt((x2 - x1)**2 + (y2 - y1)**2)

        if dist > threshold:
            count += 1
            if count >= sustain_frames:
                return smoothed[i]["frame"]
        else:
            count = 0

    return None


Why this matters:

Noise moves once

Real movement persists

This change alone removes most false positives.

4️⃣ HARDEN HEAD DRIFT (same logic)
🔴 Replace head_drift_detected with this
def head_drift_detected(frames, threshold=0.02):
    if len(frames) < 6:
        return False

    nose_positions = average_position(frames, "nose", window=3)

    start_x, start_y = nose_positions[0]["pos"]
    end_x, end_y = nose_positions[-1]["pos"]

    drift = np.sqrt((end_x - start_x)**2 + (end_y - start_y)**2)
    return drift > threshold


Now head drift only triggers on real balance loss, not camera shake.
